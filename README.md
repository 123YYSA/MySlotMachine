# 🎰 C++ 老虎机模拟器 (Boss Edition)

这是我使用 C++ 和 Windows API 纯手工打造的老虎机游戏。

## 🎮 游戏特色
- **完全开源**：代码逻辑清晰，适合新手学习。
  - 内置精心调教的 **10% 胜率**，保证庄家稳赚不赔！
  - **5% 极低大奖率**：想中 777？那是做梦！
  - **柠檬平权**：柠檬和苹果出现概率五五开。
- **视觉效果**：支持透明贴图和自动缩放。

## 🛠️ 如何运行
1. 确保所有图片文件 (“.bmp”) 和“.exe”在同一个文件夹内。
2. 无需编译，直接双击 “老虎机.exe”即可开始“赚钱”！
